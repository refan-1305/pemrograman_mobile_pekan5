# Flutter Login & Register App

## Deskripsi
Project ini merupakan aplikasi mobile sederhana yang dibuat menggunakan Flutter.  
Aplikasi ini terdiri dari halaman login, register, profile, serta sistem autentikasi pengguna menggunakan database lokal dengan validasi input.

---

## Fitur Utama
- Login dengan validasi NIM & password  
- Register akun baru  
- Penyimpanan data user ke database lokal (SQLite)  
- Validasi input:
  - Field tidak boleh kosong  
  - Password minimal 6 karakter  
  - Konfirmasi password harus sama  
- Reset password sederhana  
- Halaman profile pengguna  
- Logout kembali ke halaman login
